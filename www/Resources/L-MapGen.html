Ultra Cogent - Map Generator

This is a work in progress, don't expect much for now. When it's done it should be easy to generate realistic map that make sense, and edit it.

Tips:
- The buttons may get covered by the gradient if you scroll down. If they do not work, try scrolling up a bit more.
- If you work on a large map, full screen mode gives you more vertical space. (F11 toggles full screen mode in every browser I know. Assuming Windows or Linux on PC or laptop...)
- The smallest unit is a triangle... (Triangles fit nicely on XY coordinate system unlike hexes...)
- Rows and Columns are calculated in large hexes (the largest unit). Rows and Columns both start from 1, however Rows increment by 1, while columns increment by 2 so there's always an odd number of columns.
