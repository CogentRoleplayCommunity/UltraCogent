Project: Archiver

Halting/Archiving pages

To halt a project page content >> Run: Archiver.sh -h fielname
To archive a project page content >> Run: Archiver.sh fielname

To halt an article page content >> Run: Archiver.sh fielname

To unarchive/resume project/archive page >> Run: Archiver.sh -u fielname
To unarchive/resume project/archive page >> Run: Archiver.sh -u fielname1 fielname2 fielname3 ...
(Archiver.sh works with multiple files...)
